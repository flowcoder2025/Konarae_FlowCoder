# Accessibility (a11y) Checklist

## 기본
- [ ] WCAG 2.1 Level AA
- [ ] 색상 대비 4.5:1 이상
- [ ] 모든 인터랙션 키보드 접근 가능

## 포커스
- [ ] Tab 순서 논리적 (좌→우, 상→하)
- [ ] 포커스 표시 명확 (`focus-visible:ring-2`)
- [ ] 모달/드롭다운 포커스 트랩

## 폼
- [ ] 모든 Input에 `<label>` 연결
- [ ] 에러 시 `aria-invalid="true"`
- [ ] 에러 메시지에 `aria-describedby`
- [ ] 필수 필드 `aria-required="true"`

## 이미지/아이콘
- [ ] 의미 있는 이미지에 `alt`
- [ ] 장식용 아이콘 `aria-hidden="true"`
- [ ] 아이콘 버튼에 `aria-label`

## 동적 콘텐츠
- [ ] 에러 알림 `role="alert"`
- [ ] 상태 변경 `aria-live="polite"`
- [ ] 로딩 `aria-busy="true"`

## 금지
- [ ] `outline: none` 단독 사용 안 함
- [ ] `tabindex > 0` 사용 안 함
- [ ] 색상만으로 정보 전달 안 함
